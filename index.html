獎金比例為平台實收佣金之分配比例，每季依實際結算成交佣金總額排名，次季生效。

第一條辦法目的

為鼓勵合作仲介積極推廣平台銷售案件，建立公平、透明及長期合作之獎勵制度，特訂定本辦法。

第二條適用對象

凡與譽誠聯合銷售平台完成合作登錄之仲介公司、經紀人員、業務夥伴及聯合銷售合作人員，均適用本辦法。

第三條成交佣金分配原則

  1. 平台銷售案件之成交佣金，依建設公司實際支付平台之銷售佣金計算。
  2. 合作仲介之基本獎金比例為平台實收佣金之 50%。
  3. 獎金比例依平台菁英夥伴制度進行調整。
  4. 所有獎金均以實際收款金額為計算基礎。

第四條成交認定標準

符合下列條件後始認定為有效成交案件：

  1. 買賣契約完成簽署。
  2. 買方完成約定付款。
  3. 貸款核准及對保完成。
  4. 銀行完成撥款。
  5. 完成交屋程序。
  6. 建設公司確認成交成立。
  7. 平台取得建設公司佣金結算資料。

第五條佣金發放原則

  1. 平台佣金來源為建設公司支付之銷售佣金。
  2. 建設公司完成佣金支付後，平台始進行合作仲介佣金結算。
  3. 原則於平台收到建設公司佣金後次月完成發放。
  4. 若建設公司尚未完成佣金支付或核銷程序，平台得順延發放時間。
  5. 仲介應依規定提供請款資料、發票或相關憑證後始得辦理領款。

第六條菁英夥伴獎勵制度 暨 第七條季度排名制度

每季依實際結算成交佣金總額進行排名，等級與獎金比例如下：

等級季度排名獎金比例
鑽石夥伴第 1 名60%
白金夥伴第 2 名57%
金牌夥伴第 3 名55%
銀牌夥伴第 4 名53%
標準夥伴—50%

第八條等級適用期間

  1. 每季統計一次業績排名。
  2. 次季起適用新的獎金比例。
  3. 等級有效期間為一個季度。
  4. 新加入合作夥伴於首次排名公告前均適用標準夥伴資格。

第九條年度榮譽獎勵

年度累積成交業績前三名者，可獲得：

  1. 年度績優夥伴表揚。
  2. 特別獎勵金。
  3. 海外參訪或旅遊獎勵。
  4. 區域合夥人優先資格。
  5. 新案優先銷售資格。

實際獎勵內容由平台另行公告。

第十條客戶歸屬原則

  1. 客戶須完成平台報備程序。
  2. 客戶歸屬以平台系統紀錄為準。
  3. 未完成報備者不列入佣金認定。
  4. 客戶歸屬發生爭議時，以平台紀錄及相關證明文件作為最終認定依據。

第十一條解約案件處理

有下列情形之一者，不列入成交獎金計算：

  1. 買方主動解約。
  2. 買方違約。
  3. 貸款未核准且無法補足差額。
  4. 建設公司依法解除契約。
  5. 其他導致交易無法完成之情形。

如已發放獎金，平台有權追回已發放之佣金。

第十二條違規處理

合作仲介如有下列情形之一，平台得取消獎金資格：

  1. 不實報備客戶。
  2. 惡意搶客。
  3. 私下成交。
  4. 散布不實資訊。
  5. 損害平台或建設公司商譽。
  6. 違反合作規範。

情節重大者，平台得終止合作資格。

第十三條附則

  1. 本辦法如有未盡事宜，由譽誠聯合銷售平台另行公告補充。
  2. 平台保留獎勵制度調整及最終解釋權。
  3. 合作夥伴加入平台即視為同意本辦法全部內容。

    隱私權政策

    譽誠聯合銷售平台（下稱「本平台」）重視您的個人資料保護。本政策說明我們如何蒐集、使用與保管資料。

    一、蒐集的資料

    仲介合作申請時提供的姓名、公司、電話、Email 與經紀人證號；客戶報備時登錄的客戶姓名與聯絡電話；以及您與本平台聯繫時提供的資訊。

    二、使用目的

    資料僅用於聯合銷售業務之運作：仲介身分審核、客戶報備歸屬判定、帶看安排與成交結算。本平台不會將資料出售或提供予無關第三方。

    三、保管與存取

    資料儲存於具存取控制的雲端資料庫。報備客戶資料僅報備仲介本人與平台管理員可見，其他仲介無法查看。

    四、您的權利

    您可隨時聯繫 yuchen.web.service@gmail.com 查詢、更正或請求刪除您的個人資料。

    使用本平台即表示您同意以下條款。

    一、平台性質

    本平台為建商成屋餘屋之聯合銷售資訊與報備工具。網站所示建案資訊（含價格、戶別、完工時程）僅供參考，實際以合約與現場公告為準。

    二、仲介報備規則

    客戶歸屬以報備時間先後為準；報備成功後於有效期內受保障。仲介應確保報備資料真實正確，虛偽報備經查證得終止合作。

    三、帳號與資料

    仲介帳號僅供本人使用，不得轉借。因帳號外流造成之損失由使用者自行承擔。

    四、條款修訂

    本平台得視營運需要修訂本條款，修訂後公告於本頁面。如有疑問請洽 yuchen.web.service@gmail.com。

    第一條　合作目的

    建設公司為提升銷售效率、擴大銷售通路、縮短銷售週期及增加市場曝光度，委由譽誠聯合銷售平台整合仲介通路及銷售資源，共同推動建案銷售，特訂定本合作條款。

    第二條　合作方式

    雙方同意以下列方式之一進行合作：

    第一項　專任委託銷售

    1. 建設公司委託譽誠聯合銷售平台為本案唯一銷售單位。
    2. 合作期間內，建設公司不得再委託其他代銷公司、聯合銷售平台、仲介公司或第三方銷售機構銷售本案。
    3. 建設公司自行成交之客戶，亦應依雙方約定辦理客戶認定及佣金計算。
    4. 平台享有本案專任銷售權。

    第二項　專任聯合銷售合作 （建議採用）

    1. 建設公司保留接待中心、自有客戶及親友客戶之銷售權利。
    2. 建設公司同意將所有外部仲介通路、聯合銷售通路及跨品牌銷售通路統一由譽誠聯合銷售平台管理。
    3. 合作期間內，建設公司不得再委託其他聯合銷售平台、代銷公司或跨店聯賣組織辦理相同產品之聯合銷售。
    4. 平台享有本案對外聯合銷售之專任合作權。

    第三條　合作期間

    1. 合作期間以六個月至十二個月為原則。
    2. 合作期間內，雙方應善盡合作義務。
    3. 合作期滿得經雙方同意續約。

    第四條　履約保證專戶佣金提撥機制

    1. 本合作案必須採用履約保證專戶佣金提撥制度。
    2. 建設公司同意授權履約保證機構、指定代書或相關履約單位辦理佣金提撥作業。
    3. 買方完成銀行撥款時，依約定比例直接提撥銷售佣金至譽誠廣告有限公司指定帳戶。
    4. 建設公司應配合簽署相關授權文件。
    5. 建設公司不得拒絕或延遲辦理佣金提撥。
    6. 履約保證專戶佣金提撥機制為雙方合作之必要條件。

    第五條　客戶報備及保護制度

    1. 平台得建立客戶報備制度。
    2. 平台合作仲介、區域合夥人及平台自有廣告客戶均應完成報備程序。
    3. 客戶歸屬以平台系統紀錄為準。
    4. 客戶保護期間由雙方約定，建議為 90 日至 180 日。
    5. 保護期間內成交之客戶均應認列為平台成交案件。

    第六條　成交認定標準

    符合下列條件後即視為成交案件：

    1. 買賣契約完成簽署。
    2. 買方完成應付款項。
    3. 貸款核准及對保完成。
    4. 銀行完成撥款。
    5. 完成交屋程序。

    符合上述條件即應辦理佣金提撥及結算。

    第七條　平台客戶保護條款

    1. 平台報備客戶不得轉由其他通路成交。
    2. 平台廣告取得之客戶不得規避平台成交認定。
    3. 平台安排帶看之客戶不得轉由第三方接手成交。
    4. 建設公司不得以自行成交名義規避平台應得佣金。

    第八條　專任合作限制條款

    合作期間內，建設公司不得：

    1. 委託其他聯合銷售平台。
    2. 委託其他跨品牌聯賣組織。
    3. 委託其他代銷公司進行相同產品聯賣。
    4. 以其他名義規避本合作契約。

    如有違反，平台得要求終止合作並保留相關求償權利。

    第九條　合作終止後之客戶保護

    1. 合作終止前已完成報備之客戶，仍受客戶保護條款保障。
    2. 合作終止後六個月內成交之已報備客戶，仍應認列為平台成交案件。
    3. 建設公司仍應依約辦理佣金提撥及結算。

    第十條　合作精神

    譽誠聯合銷售平台秉持「共享案源、共享客源、共享成交、共享獲利」，透過專任聯合銷售制度、客戶報備制度及履約保證專戶佣金提撥制度，建立建設公司、仲介夥伴及平台三方共贏之合作模式。
